# Kinta Health
Web application made with MERN stack to be used by hospitals and similar medical entities.
It has a frontend based in React and some icons and pre-made components and a node-express API.
There's a user login and register system where the user can check all the doctors in the team, and request for appointments.
A user can also apply for a doctor user role and have other features. There is also an admin user role to manage everything.
